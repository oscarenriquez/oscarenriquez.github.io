## Context

Modernization projects matter when they improve both customer experience and the speed at which teams can ship.

This work moved a legacy portal toward a more maintainable full-stack architecture while still supporting live product needs.

## Impact

- Improved user experience and frontend performance
- Reduced system complexity compared with the legacy portal model
- Increased delivery speed for new features

## Contribution

I worked across the frontend and backend stack, supported the migration effort, and delivered business-facing features for product areas such as home security.
