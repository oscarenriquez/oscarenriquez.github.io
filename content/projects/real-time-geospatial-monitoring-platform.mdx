## Context

Real-time operational platforms need to make event data understandable immediately, especially when location is part of the decision.

This project focused on translating live activity into a usable visual system instead of leaving users to interpret raw signals.

## Impact

- Enabled real-time monitoring of incidents on maps
- Improved geospatial analysis for operational scenarios
- Supported faster interpretation of live event activity

## Contribution

I developed the core visualization module, contributed to the backend services, and helped build the analytics capabilities that made the platform more useful than a simple map display.
