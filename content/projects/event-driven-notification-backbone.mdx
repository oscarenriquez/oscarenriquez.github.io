## Context

Financial notification systems need to be reliable without slowing down the transaction flows that trigger them.

This project focused on moving notification processing into an asynchronous architecture that improved resilience while keeping operational behavior understandable.

## Impact

- Supported reliable high-volume processing for payment and donation events
- Reduced synchronous dependencies across critical systems
- Improved resilience and scalability for notification delivery
